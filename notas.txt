Arquitetura MVC com Node.js + Express

O MVC (Model, View, Controller), é um modelo de desenvolvimento de software
que tem como objetivo modular e tornar manutenível um projeto de software.

Com Node.js + Express, organizamos o projeto geral dentro de uma pasta 'src',
que possuem os diretórios 'models', 'views' e 'controllers'.

Em 'models' estarão todos os arquivos de models referente a base de dados;
Em 'views', as estruturas visuais dos dados processados a serem exibidas para o usuário;
Em 'controllers', os arquivos responsáveis por gerenciar as requisições do usuário, processar essas informações e devolver nas views.

No MVC, o usuário faz requisição por um recurso. Esses recursos são enviados para o controller, que processa as informações
necessárias e então as envia para uma view e a renderiza de volta para o usuário como resposta.
